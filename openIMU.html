Old Control Hub

As the 2023-24 season control hub has a faulty chip, using a previous control hub can solve any issues you might be having. We have been using our control hub from last season and it has been working flawlessly. However if you do not have an old control hub, finding a previous generation might be hard, not to mention the cost. See the next step on how to install an external IMU.
